Second · Auth Tester

Frontend for API Auth Validation

Use this page to test registration, email verification, login/logout, /me, password reset, and seller upgrade against your running API.

How to use

  1. Register a user.
  2. Request and verify email token.
  3. Login and let this page store your access token in memory.
  4. Call protected endpoints like /api/auth/me.

Token storage is in-memory only for testing. Refreshing page clears it.
